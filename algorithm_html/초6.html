2702번 초6 수학

#수학 #브루트포스 알고리즘 #정수론 #사칙연산


  • Python

  • Java

  • JavaScript

  • Dart

출처: https://www.acmicpc.net/problem/2702

주어지 두 정수를 a, b라 할때, 최소공배수와 최대공약수를 구해야한다.
유클리드 호제법을 이용하면 두 수의 최대공약수를 구할 수 있다.

* 유클리드 호제법이란, 두 자연수 A, B가 있을 때
1) A와 B의 최대공약수는 'A를 B로 나눈 나머지'와 'B'의 최대공약수와 같다.
2) A와 B의 최대공약수는 'A를 B로 뺀 값'과 'B'의 최대공약수와 같다.
(A와 B가 같아질 때까지 큰 수를 작은 수로 빼며 반복)

여기서는 1번 방법을 이용해 적은 연산횟수로 빠르게 최대공약수를 구할 수 있었다.
최소공배수는 A와 B의 곱셈에서 최대공약수를 나눈 값이 된다.
