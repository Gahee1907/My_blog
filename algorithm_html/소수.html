2581번 소수

#수학 #정수론 #소수 판정


  • Python

  • Java

  • JavaScript

  • Dart

출처: https://www.acmicpc.net/problem/2581

M과 N이 주어질 때 M이상 N이하의 자연수 중 소수인 것을 모두 골라 이들 소수의 합과 최솟값을 찾는 문제이다.

소수는 1과 자기 자신을 약수로 가지는 수이다.
i와 j가 같을때, 자기 자신을 약수로 가지는 상황이다. 만약 i와 j가 같다면 소수로 판단해서 sum에 합산한다.
i와 j의 몫이 '0'이라면 반복문을 빠져나오고, 소수가 없다면 '-1'을 출력시켜주었다.
