1065번 한수

#수학 #브루트포스 알고리즘


  • Python

  • Java

  • JavaScript

  • Dart

출처: https://www.acmicpc.net/problem/1065

>

등차수열은 연속된 두 개의 수의 차이가 일정한 수열이다.
N이 주어졌을 때, 1보다 크거나 같고, N보다 작거나 같은 한수의 개수를 출력하는 문제이다.

각 자리수의 차이값이 같은 경우가 한수이다.
두 자리수 이하는 등차수열인지 판단할 수 없기 때문에 한수로 간주한다.
