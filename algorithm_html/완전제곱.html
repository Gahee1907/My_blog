1977번 완전제곱수

#수학 #구현 #브루트포스 알고리즘


  • Python

  • Java

  • JavaScript

  • Dart

출처: https://www.acmicpc.net/problem/1977

M과 N이 주어질 때, ( M <= X <= N ) 의 자연수 중에서 완전제곱수인 것들의 합과 최솟값을 구해야한다.

주어지는 수가 10,000이하이기 때문에 최대 100까지만 확인하면 된다.
M과 N사이에서 완전제곱값을 찾은 후 합산하고, min에 갱신된 최솟값을 출력했다.
