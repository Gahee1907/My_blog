4153번 직각삼각형

#수학 #기하학 #피타고라스정리


  • Python

  • Java

  • JavaScript

  • Dart

출처: https://www.acmicpc.net/problem/4153


직각삼각형은 피타고라스의 공식에 따라서 두변의 길이의 제곱의 합과 가장 긴변의 제곱의 값이 같다고 정의 된다.

따라서 각 변을 제곱하고, 긴 변의 제곱과 나머지 두변의 제곱의 합이 같은 세가지 경우를 넣어 같으면 'right',
다르면'wrong'을 출력했다.
