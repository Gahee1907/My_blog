1094번 막대기

#수학 #비트마스킹


  • Python

  • Java

  • JavaScript

  • Dart

출처: https://www.acmicpc.net/problem/1094

지민이가 가지고 있는 막대의 길이를 모두 더한다.
처음에는 64cm 막대 하나만 가지고 있다. 이때, 합이 X보다 크다면, 아래와 같은 과정을 반복한다. (X는 23)

1. 가지고 있는 막대 중 길이가 가장 짧은 것을 절반으로 자른다.
>> 가지고 있는 막대기(64cm)가 23cm보다 크기 때문에 64cm를 절반으로 자름 (32cm)
>> 가지고 있는 막대기가(32cm) 23cm보다 크기 때문에 32cm를 절반으로 자름 (16cm)

2. 만약, 위에서 자른 막대의 절반 중 하나를 버리고 남아있는 막대의 길이의 합이 X보다 크거나 같다면,
위에서 자른 막대의 절반 중 하나를 버린다.
>> 다시 자름(16 + 8 = 24cm)
>> 하나 보관하고 남은 하나 자름(16 + 4 = 20cm)
>> 하나 보관하고 남은 하나 자름(16 + 4 + 2 = 22cm)
>> 하나 보관하고 남은 하나 자름(16 + 4 + 2 + 1 = 23cm)

이제, 남아있는 모든 막대를 풀로 붙여서 Xcm를 만든다.
16cm, 4cm, 2cm, 1cm 총 4개
