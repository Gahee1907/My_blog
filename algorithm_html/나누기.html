#수학 #브루트포스 알고리즘


  • Python

  • Java

  • JavaScript

  • Dart

출처: https://www.acmicpc.net/problem/1075

n을 f로 나누어 떨어진다는 말은 n % f = 0이고 즉, n은 f의 배수가 된다는 것이다.
두 자리에서 만들 수 있는 값은 총 100개이다.
가장 낮은 수에서부터 차례대로 살피면 조건에 부합하는 첫 번째 값이 최소값이 된다.
